# Data Analyst Internship - Task 1

## Objective
Clean and preprocess a raw dataset by:
- Handling missing values
- Removing duplicates
- Standardizing text values
- Converting date formats
- Renaming columns
- Fixing data types

## Steps Performed
1. Load dataset using Pandas
2. Check missing values with isnull()
3. Fill or remove missing values
4. Remove duplicates
5. Standardize text columns
6. Convert date columns to datetime
7. Rename columns to lowercase with underscores
8. Export cleaned dataset

## Deliverables
- cleaning_script.py
- cleaned_dataset.csv
- interview_answers.md
